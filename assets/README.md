# Placeholder Assets

## hero.jpg

**Requerido**: Imagen de 1920x1080px, máx 500KB

### Opciones:
1. **Propia**: Usa una foto real de los novios
2. **Placeholder**: Descarga alguna imagen libre de derechos

### Recomendaciones técnicas:
- Formato: JPG o WebP (WebP es más ligero)
- Resolución: 1920x1080px (full HD)
- Peso: < 500KB para LCP < 2.5s
- Orientación: Horizontal (16:9)

### Fuentes de imágenes gratuitas:
- Unsplash: unsplash.com/s/photos/wedding
- Pexels: pexels.com/es/wedding
- Pixabay: pixabay.com/es/images/search/wedding/

## favicon.ico

**Opcional**: Icono para la pestaña del navegador

### Generar:
1. Ve a https://favicon.io/
2. Sube una imagen o genera desde emoji
3. Download el .ico
4. Coloca en assets/
